Resources API¶

Dataset¶

POST (string: server_name)/nd/resource/dataset/¶
Synopsis:

Create a dataset
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is cloud.neurodata.io
Status Codes:
  • 201 Created – Resource created
  • 400 Bad Request – Bad Request. Incorrect syntax
  • 403 Forbidden – Forbidden. Insufficient permissions.

Example Request:

POST /nd/resource/kasthuri11/ HTTP/1.1
Host: cloud.neurodata.io
Content-Type: application/json

{
  "dataset_name" : "kasthuri11",
  "ximagesize" : 2000,
  "yimagesize" : 2000,
  "zimagesize" : 100,
  "xvoxelres" : 5.0,
  "yvoxelres" : 5.0,
  "zvoxelres" : 10.0,
  "public" : 1
}

Example Responses:

HTTP/1.1 201 Created
Content-Type: text/plain

HTTP/1.1 400 BadRequest
Content-Type: text/plain

HTTP/1.1 403 Forbidden
Content-Type: text/plain
