Project Info API¶

JSON Informatio Service¶

GET¶

GET (string: server_name)/nd/sd/(string: token_name)/info/¶
Synopsis:

Get all metadata associated with a project that the specified token refers to, including metadata for the dataset it points to, as well as each channel it contains in JSON format.
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is cloud.neurodata.io
  • token_name (string) – Token Name in NeuroData.
Status Codes:
  • 200 OK – No error
  • 403 Forbidden – Forbidden
  • 404 Not Found – Error in the syntax

Example Request:

GET /nd/sd/kasthuri11/info/ HTTP/1.1
Host: cloud.neurodata.io

HDF5 Information Service¶

Synopsis:

Get all metadata associated with a project that the specified token refers to, including metadata for the dataset it points to, as well as each channel it contains in HDF5 format.
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is cloud.neurodata.io
  • token_name (string) – Token Name in NeuroData.
Status Codes:
  • 200 OK – No error
  • 403 Forbidden – Forbidden
  • 404 Not Found – Error in the syntax

Example Request:

GET /nd/sd/kasthuri11/projinfo/ HTTP/1.1
Host: cloud.neurodata.io

Example Response:

HTTP/1.1 200 OK
Content-Type: application/hdf5
