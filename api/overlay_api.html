Overlay API¶

Cutut Service¶

GET XY Overlay Slice Cutout¶

POST (string: host_server_name)/nd/overlay/(float: alpha_value)/(string: first_server_name)/(string: first_token_name)/(string: first_channel_name)/(string: second_server_name)/(string: second_token_name)/(string: second_channel_name)/xy/(int: resolution)/(int: min_x),(int: max_x)/(int: min_y),(int: max_y)/(int: z_slice)/(int: time_slice)/¶
Synopsis:

Get a XY Slice Cutout
Parameters:
  • host_server_name (string) – Host Server Name in NeuroData. In the general case this is openconnecto.me.
  • first_server_name (string) – First Server Name in NeuroData. In the general case this is openconnecto.me.
  • first_token_name (string) – First Token Name in NeuroData.
  • first_channel_name (string) – First Channel Name in NeuroData. Optional. If missing will use default channel for the token.
  • second_server_name (string) – Second Server Name in NeuroData. In the general case this is openconnecto.me.
  • second_token_name (string) – Second Token Name in NeuroData.
  • second_channel_name (string) – Second Channel Name in NeuroData. Optional. If missing will use default channel for the token.
  • resolution (int) – Resolution for the data
  • min_x (int) – Minimum value in the xrange
  • max_x (int) – Maximum value in the xrange
  • min_y (int) – Minimum value in the yrange
  • max_y (int) – Maximum value in the yrange
  • z_slice (int) – Z-slice value
  • time_slice (int) – Minimum value in the timerange. Optional. Only used for timeseries channels.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

GET YZ Overlay Slice Cutout¶

POST (string: host_server_name)/nd/overlay/(float: alpha_value)/(string: first_server_name)/(string: first_token_name)/(string: first_channel_name)/(string: second_server_name)/(string: second_token_name)/(string: second_channel_name)/yz/(int: resolution)/(int: x_slice)/(int: min_y),(int: max_y)/(int: min_z),(int: max_z)/(int: time_slice)/¶
Synopsis:

Get an overlay YZ slice cutout
Parameters:
  • host_server_name (string) – Host Server Name in NeuroData. In the general case this is openconnecto.me.
  • first_server_name (string) – First Server Name in NeuroData. In the general case this is openconnecto.me.
  • first_token_name (string) – First Token Name in NeuroData.
  • first_channel_name (string) – First Channel Name in NeuroData. Optional. If missing will use default channel for the token.
  • second_server_name (string) – Second Server Name in NeuroData. In the general case this is openconnecto.me.
  • second_token_name (string) – Second Token Name in NeuroData.
  • second_channel_name (string) – Second Channel Name in NeuroData. Optional. If missing will use default channel for the token.
  • resolution (int) – Resolution for the data
  • x_slice (int) – X-slice value
  • min_y (int) – Minimum value in the yrange
  • max_y (int) – Maximum value in the yrange
  • min_z (int) – Minimum value in the zrange
  • max_z (int) – Maximum value in the zrange
  • min_time (int) – Minimum value in the timerange. Optional. Only used for timeseries channels.
  • max_time (int) – Maximum value in the timerange. Optional. Only used for timeseries channels.
Form Parameters:
 
  • CUTOUT – HDF5 group, Post data
  • CHANNELTYPE – HDF5 group, Channel type(image, annotation, probmap, timeseries)
  • DATATYPE – HDF5 group, Data type(uint8, uint16, uint32, rgb32, rgb64, float32)
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format
