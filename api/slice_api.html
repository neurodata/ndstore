Slice API¶

NOTE: These APIs do not return the orginal data. If you want the orginal raw data then please use the Data APIs.

GET XY Slice Cutout¶

GET (string: server_name)/nd/sd/(string: token_name)/(string: channel_name)/xy/(int: resolution)/(int: min_x),(int: max_x)/(int: min_y),(int: max_y)/(int: z_slice)/(int: time_slice)/¶
Synopsis:

Download a single image of a specified canonical plane, XY, and specified channel, resolution and bounds. Your browser can load these images.
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is cloud.neurodata.io.
  • token_name (string) – Token Name in NeuroData.
  • channel_name (string) – Channel Name in NeuroData. Optional. If missing will use default channel for the token.
  • resolution (int) – Resolution for the data
  • min_x (int) – Minimum value in the xrange
  • max_x (int) – Maximum value in the xrange
  • min_y (int) – Minimum value in the yrange
  • max_y (int) – Maximum value in the yrange
  • z_slice (int) – Z-slice value
  • time_slice (int) – Minimum value in the timerange. Optional. Only used for timeseries channels.
Status Codes:
  • 200 OK – No error
  • 403 Forbidden – Forbidden
  • 404 Not Found – Error in the syntax or file format

Example Request:

GET /nd/sd/kasthuri11/image/xy/0/9000,9500/12000,12500/50/ HTTP/1.1
Host: cloud.neurodata.io

Example Response:

HTTP/1.1 200 OK
Content-Type: image/tiff

GET YZ Slice Cutout¶

GET (string: server_name)/nd/sd/(string: token_name)/(string: channel_name)/yz/(int: resolution)/(int: x_slice)/(int: min_y),(int: max_y)/(int: min_z),(int: max_z)/(int: time_slice)/¶
Synopsis:

Download a single image of a specified canonical plane, YZ, and specified channel, resolution and bounds. Your browser can load these images.
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is cloud.neurodata.io.
  • token_name (string) – Token Name in NeuroData.
  • channel_name (string) – Channel Name in NeuroData. Optional. If missing will use default channel for the token.
  • resolution (int) – Resolution for the data
  • x_slice (int) – X-slice value
